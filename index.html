About
Blox is a simple block-pushing game made by Eric using Momo. Blox's source code is freely available on GitHub. You can check out Eric's other games here.

Instructions
Use the arrow keys to move around, the "R" key to restart a puzzle if you get stuck, and the "Z" key to undo moves. Push the blocks into the goals to win.
